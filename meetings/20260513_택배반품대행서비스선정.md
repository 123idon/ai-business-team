# 회의록
일시: 2026-05-13 12:59
주제: 생활 불편함 포착형 서비스 사업 아이템 선정
참석자: 김전략(CEO), 이마케팅(CMO), 박재무(CFO), 최기술(CTO), 정법무(CLO), 한전략(CSO)

## 1. 회의 목표
쓰레기 수거 서비스처럼 일상의 귀찮음을 해결하는 서비스 비즈니스 후보 도출 및 최종 방향 선정

## 2. 사업 모델 성립 조건 (합의)
- 반복성: 일회성이면 사업 안 됨
- 표준화 가능성: 매번 다르면 단가 못 맞춤
- 디지털 접점: 앱/카카오로 주문 가능해야 함
- 지역 밀도: 특정 지역 집중 → 인력 효율 확보

## 3. 검토 후보 아이템
1. 택배 반품 대행 ← 권고안
2. 소형 폐가전/가구 처리 대행 (플랫폼 구조) ← 중기 확장
3. 냉장고 정리/유통기한 식재료 처분 대행
4. 재활용품 분리 대행
5. 아파트 단지 기반 종합 생활 편의 플랫폼

## 4. 최종 결론
**권고안: 택배 반품 대행 서비스**
- 이커머스 성장 = 반품 건수 지속 증가
- 경쟁 강도가 세탁·청소 대비 낮음 (추정)
- 법적 리스크 낮음 (단순 운반 대행)
- 수익 구조: 건당 2,000~5,000원 + 구독 월정액

**중기 확장: 소형 생활폐기물 처리 플랫폼**
- 택배 반품으로 고객·인력 기반 확보 후 확장

## 5. 액션 아이템
1. [1주 내] 타깃 아파트 단지 주민 20~30명 수요 설문 / CEO+CMO
2. [1주 내] 경쟁 서비스 3개 직접 체험 후 차별화 포인트 문서화 / CSO
3. [2주 내] 아파트 단지 1개 기준 손익분기점 산출 / CFO

## 6. 추가 검토 필요
- 주요 택배사 API 연동 가능성
- 아파트 단지 내 영업/홍보 허가 요건
- 반품 분실/파손 시 책임 소재 (CLO 검토)

# 회의록
일시: 2026-05-13 13:29
주제: 문제 해결형 B2B+B2C 사업 재탐색 — 평판·신뢰 검증 플랫폼
참석자: 김전략(CEO), 이마케팅(CMO), 박재무(CFO), 최기술(CTO), 정법무(CLO), 한전략(CSO)

## 1. 탐색 조건
- 문제 해결형 (불편함·비효율 제거)
- B2B + B2C 동시 수익 가능
- AI가 도구로 활용 가능 (경쟁자 아님)
- 특정 배경 불필요

## 2. 검토 후보 3개

### 후보 A. 계약·전자서명 자동화
- 해외: DocuSign, PandaDoc / 국내: 모두싸인 (선점)
- 선발 주자 극복이 핵심 과제

### 후보 B. 서비스 가격 투명화
- 해외: Thumbtack, Bark.com / 국내: 숨고·크몽 (선점)
- 차별화 약함, 수익화 가장 어려움

### 후보 C. 평판·신뢰 검증 플랫폼 ← 팀 결론
- 해외: Checkr($57억), Certn, Veremark
- 국내 공백: 대기업 배경조사만 있음, 개인·중소기업 대상 없음

## 3. 최종 결론: 평판·신뢰 검증 플랫폼 (Trust Layer)

### 핵심 정의
"사람과 업체의 신뢰를 빠르고 쉽게 검증해주는 플랫폼"

### 해결하는 문제
- 개인: 프리랜서·중고거래·집주인 신뢰 여부
- 기업: 입사자·파트너사·외주업체 레퍼런스 체크
- AI 시대에 두 문제 모두 더 심각해짐 (AI 위조 증가)

### 핵심 설계 (법적 안전 구조)
피검증자 직접 동의·정보 제출 → AI+사람 검증·요약 → 검증 배지/레포트 발급

### 수익 모델
- B2C: 개인 조회 건당 3,000~1만원
- B2B: 기업 구독 월 30~100만원 + 건당 레포트 5~15만원

### 후보 1(인증)과의 관계
후보 1의 욕구("진짜임을 증명")를 '작품 인증'이 아닌 '사람 신뢰 검증'으로 재해석
→ 법적·기술적 문제 해소 + AI 시대에 오히려 수요 증가

## 4. 차별화
- 기존 배경조사: 대기업만, 비쌈, 느림 (100만원+)
- 이 플랫폼: 개인·중소기업도 가능, 저렴, AI로 빠름

## 5. 액션 아이템
1. [즉시] 창업자 방향 확정
2. [1주] 국내 배경조사·신원확인 법적 진입 가능성 확인 (CLO)
3. [2주] 잠재 고객 인터뷰 설계 (CMO)

## 6. 핵심 전제 검증
- 민간 플랫폼의 개인 신원 검증 법적 범위
- '당사자 동의 제출' 방식 법적 안전성 (CLO 1순위)
